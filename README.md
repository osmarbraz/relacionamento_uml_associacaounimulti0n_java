# Exemplo de implementação de relacionamento associação unidirecional e multiplicidade 0..* e papel em Java.

## Contextualização

- Programa em Java que demostra a implementação do relacionamento associação unidimensional "usa" com multiplicidade 0..* e papel.<br>
- Uma associação é uma conexão entre classes e representam as relações entre os objetos.<br>
- Associações são representadas em um diagrama de classe através de uma linha conectando as classes associadas.<br>
- Os dados podem fluir em uma ou em ambas as direções através do link.<br>
- O projeto foi desenvolvido no NetBeans deve ser chamado relacionamento_uml_associacaounimulti0n_java.<br>
- Utiliza o Apache Maven para a automatização da construção.<br>
